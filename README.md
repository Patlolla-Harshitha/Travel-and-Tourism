# Travel and Tourism Expense Predictor

## Project Overview
This project helps users estimate travel expenses based on the starting location, destination, and mode of transport.

## Features
- Enter starting location and destination
- Select mode of transport
- Calculate estimated travel expenses
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
